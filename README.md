# Tengja

**Iceland's Network Pulse** — real-time submarine cable visualization.

Tengja is an interactive web application that visualizes Iceland's submarine cable infrastructure and network connectivity status in real-time. Explore the undersea cables that connect Iceland to the world, with animated data flows and live status monitoring.

## Tech Stack

- Next.js 16 (App Router, Turbopack)
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Geist font family

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/
├── page.tsx          # Main visualization page
├── dashboard/        # Network dashboard view
├── api/              # API routes
└── ui/               # UI components

lib/                  # Utilities and data
```
